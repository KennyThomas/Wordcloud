*Data Structures and Algorithms - Assignment2019*

A Java application to Create a Word-cloud,This application is reuquired to parse a file or a URL to generate a PNG
file with the words displaying in differn't sizes , style.
This application ignores certain words from a file called ignorewords.txt.


*Main Features*
-A command-line menu that prompts the user with a suite of choices
-Read in a textFile
-Read in a URL
-Count the occurences of words
-Find the time taken to count the words
-Display the amount of chosen words
-Create a PNG image of words
-I added the feature so the user can either pick random colors or create own color that would be used

Eclipse Oxygen Java 1.8
